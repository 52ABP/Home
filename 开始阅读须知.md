# 开发环境
这里是指使用52ABP框架开发需要准备的环境和信息，会不断的进行更新。

## 课程中会使用到的技术和工具

- 语言：C#、Typescript 为主 
- 框架：.Net Core Angular 
- 开发工具：Visual Studio 2017、Visual Studio Code
- 数据库：SQL Server 2012+
- 缓存：本地缓存、Redis
- 浏览器：Chrome、Firefox 等
- FTP客户端：FileZilla等
# 企业版的52ABP说明
